# Carrier

## Overview
A **Carrier** is a transport service provider operating fleets and hubs to move goods across distances. Carriers range from hyperlocal courier companies to long-haul freight operators.

## IRI
`https://schema.beckn.org/logistics/Carrier`

## Use Cases
- Courier: FedEx, DTDC, Delhivery
- Interstate: Road freight operators
- Long Haul: Full-truck-load national freight carriers
- Express: Blue Dart, Ekart Express
